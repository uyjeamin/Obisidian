**데이터 모델링이란?** : 정보 시스템 구축의 대상이 되는 업무내용을 분석하고 약속된 표기법에 의해 표현하는 것


이렇게 분석된 모델을 가지고 실제 DB 를 생성하여 개발한다.


데이터를 추상화한 데이터 모델은 DB의 틀을 이해하고 그 이해를 바탕으로 SQL 을 효율적으로 작성할수 있다.
